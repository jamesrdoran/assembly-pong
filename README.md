# assembly-pong
![image](https://github.com/jamesrdoran/assembly-pong/assets/139739768/c702a402-fe15-4bc9-a6de-393b428aea01)

Pong game written in assembly

### movement

left paddle:
  'w' or 'W' - up
  's' or 'S' - down
right paddle:
  'o' or 'O' - up
  'l' or 'L' - down
